# Build Your Own Minus 2S Robot

## Description

The **Minus 2S** is the latest generation of our 4WD mobile robot platform, featuring a robotic gripper, a pan-tilt camera head, infrared night vision, and full compatibility with the Vigibot remote-control platform.

Designed around custom PCBs, a 2S battery architecture, and the new **Vigi UPS V3**, it offers simplified assembly, improved reliability, and enhanced power capabilities compared to the previous Minus 1S platform.

All the parts required to build the robot are available in the following kit:  
[Minus 2S Kit](https://www.robot-maker.com/shop/kits-robots/284-kit-minus-2s-284.html)
